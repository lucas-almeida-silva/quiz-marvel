# :seedling: How to run

```bash
# Clone the repository
$ git clone https://github.com/lucas-almeida-silva/quiz-marvel.git

# Go to the project folder
$ cd quiz-marvel

# Install Dependencies
$ yarn
# or npm install

# Run the application
$ yarn dev
# or npm run dev
```
Access the API at http://localhost:3000
